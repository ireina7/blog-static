1. Skeleton 是什么？

Skeleton 是一门用于文本与笔记撰写排版的编程语言 （A programming language for text typesetting）。 Skeleton 既可以用于简单的markdown文本撰写，也可以用于通用的编程（这里的编程大多指在文本排版领域）。 你可以将 Skeleton 用于：

  • 日常的个人笔记撰写
  • 复杂的文本排版与template编程
  • 使用不同的后端模块生成多种文本格式（e.g. html, pdf, markdown, etc）


2. 基本元素

Skeleton 是一门十分精简的语言。语言主体包括如下几种不可分割的最基本的元素：

  • 变量 variable
  • 数字 number
  • 文本 text

2.1 变量

在 Skeleton 中，所有的变量以 \开头。 比如我们需要写 “\”这个符号， 在Skeleton 中需要写作：\slash or \\（因为反斜杠在Skeleton 会被自动识别为变量开头）。

2.2 数字

Skeleton 支持两种形式的数字：整数（integers）和浮点数（real number）。 所有数字支持基本的算术运算。

2.3 文本

文本在Skeleton 中是第一等公民。 你可以在任何地方任意书写普通文本（与大部分编程语言不同的是Skeleton 不需要添加引号）。


3. 组合

有了基本元素，下一步的目标就是组合这些元素来构造我们需要的新元素。

3.1 文本组合

文本组合是 Skeleton 提供的最基本的组合方式之一。假设我们有了文本 A和 B， 组合两者只需要简单写作：A B。需要注意的是，当组合变量与文本时，变量与文本的连接处需要加上符号进行分隔。 比如：\Skeleton\ is greator {\Skeleton} is great。

3.2 函数组合

函数组合是 Skeleton 的重要特性。在 Skeleton 中，你可以自由组合各种函数来生成你需要的文本与样式。 函数组合（调用）的最基本语法是： 

(\f x y z...)


4. 抽象

抽象，抽象，再抽象！
函数（function）是 Skeleton 最基本的抽象元素。

4.1 函数

Skeleton 与markdown类型的标记语言最大的不同点在于 Skeleton 支持函数定义与调用。

4.2 定义

Skeleton 支持两种最基本的定义方式：变量定义与函数定义。 变量定义的格式为： 

(\set \[variable-name] [expression])

函数定义的语法只是在变量的位置引入括号调用语法（这十分直观！）： 

(\set (\[function-name] \x \y)) {
  [expression]
}


5. 面向文本对象

在 Skeleton 中，你可以认为所有的方法都是面向文本对象的， 即所有的函数的最后一个参数都是绑定待操作文本的 \self 参数。
